# Mobile Sales Performance Dashboard | Power BI

## Project Overview

The **Mobile Sales Performance Dashboard** is an interactive data analytics project built using **Microsoft Power BI** to analyze mobile sales data and generate meaningful business insights.

The dashboard provides a comprehensive view of sales performance, customer ratings, product trends, payment methods, and city-wise sales distribution through interactive visualizations and filters.

---

## Project Objective

The objective of this project is to transform raw mobile sales data into an interactive dashboard that helps understand:

- Overall sales performance
- Product and brand performance
- Customer behavior
- Sales trends over time
- Payment method distribution
- Regional sales analysis

---

## Tools & Technologies Used

- **Power BI** – Dashboard development and visualization
- **Power Query** – Data cleaning and transformation
- **DAX** – Creating calculated measures and KPIs
- **Excel** – Data source and initial data handling

---

## Dashboard Features

### Key Performance Indicators (KPIs)
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales Value

### Interactive Analysis

- Sales analysis by city  
- Daily sales trend analysis  
- Monthly quantity trend  
- Customer rating analysis  
- Payment method analysis  
- Brand-wise sales comparison  
- Mobile model performance analysis  

---

## Key Insights

- Identified top-performing mobile brands based on sales contribution
- Analyzed sales trends across different time periods
- Compared customer ratings and purchasing patterns
- Evaluated payment method preferences
- Observed city-wise sales performance distribution
